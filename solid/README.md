# S.O.L.I.D

It is a set of 5 principles to help developers to write flexible, maintainable, scalable
code

The objective here is to apply these principles using Python code. 

The references:

* Clean Architecture
* [Real Python - SOLID](https://realpython.com/solid-principles-python/#single-responsibility-principle-srp)
